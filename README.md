# Bar Asador de la Puebla

Página web del Bar Asador de la Puebla

## 🚀 Inicio rápido

### Instalación

```bash
npm install
```

### Desarrollo

```bash
npm run dev
```

El servidor de desarrollo se abrirá en `http://localhost:5173/`

### Compilar para producción

```bash
npm run build
```

Los archivos compilados se generarán en la carpeta `docs/`

### Despliegue

```bash
npm run deploy
```

Compila y despliega automáticamente a GitHub Pages

## 📁 Estructura del proyecto

```
src/
├── index.html          # Página principal
├── scss/               # Estilos SASS
├── js/                 # JavaScript
└── partials/           # Componentes HTML reutilizables
    ├── header.html
    ├── main.html
    ├── footer.html
    └── link.html

public/
└── images/             # Imágenes públicas
```
